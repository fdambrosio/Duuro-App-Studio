How to pass values to UILabel in iOS 6

Passing values to UILabel in the iOS is one of the most common things that you will when you will develop iPhone or iPad apps.
This is no issue when the UILabel is in the same viewController as the object that is changing UILabel’s value. But things can get a little tricky when you try to pass values to a UILabel that is on another viewController.
To help you achieve this thing a get an idea how this process works, we created this simple tutorial that shows you how to pass values to UILabels in iOS 6 and earlier versions.
We will create a simple Quotes App that will display a quote of  3 world’s top visionaries if you press a button with their name.
